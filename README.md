# 一键投诉法院不立案不下裁定

针对法院立案庭**不立案且不下不予立案裁定**的违法行为，一键向多个监督平台提交投诉的桌面工具。

## 功能特点

- **14 个投诉渠道**，覆盖法院、检察院、纪检监察、人大信访、信访局、司法部、政法委等
- **31 个省份**自动匹配省人大信访和省信访局网址
- **一键操作**：自动复制投诉内容到剪贴板，并打开所有选中平台的投诉页面
- **内置投诉模板**，包含法律依据，可直接使用或自行修改
- **草稿保存/加载**，支持文件导入导出，投诉历史自动记录
- **纯 Python 标准库**，无需安装任何第三方包

## 运行环境

## 使用步骤

1. **选择省份**：在顶部下拉框选择所在省份，省人大信访和省信访局将自动匹配对应网址
2. **撰写投诉内容**：在左侧文本框中撰写投诉材料（可点击"加载模板"使用预设模板）
3. **选择投诉平台**：在右侧勾选要投诉的平台（默认全选）
4. **一键投诉**：点击底部红色按钮，程序自动复制内容到剪贴板并打开所有选中的投诉网站
5. **粘贴提交**：在每个投诉页面中按 Ctrl+V 粘贴投诉内容并提交

## 界面截图

程序界面分为左右两栏：左侧为投诉内容编辑区（含模板加载、文件导入导出、草稿管理等功能），右侧为投诉平台选择区（按类别分组，支持全选/全不选）。

## 法律依据

- 《中华人民共和国民事诉讼法》第一百二十六条
- 《最高人民法院关于人民法院登记立案若干问题的规定》
- 《人民法院工作人员处分条例》

## 注意事项

1. 投诉内容请实事求是，如实反映情况
2. 建议保留好起诉材料的递交凭证（如收据、EMS回执等）
3. 部分投诉平台需要实名注册后才能提交
4. 投诉后请关注各平台的回复和处理进度
5. 本程序仅帮助打开投诉页面并复制内容，实际提交需要用户在各平台手动完成
