# 一键投诉法院不立案不下裁定 - 使用说明

## 简介

本程序是一个本地运行的桌面工具，专门针对**法院立案庭不立案且不下不予立案裁定**的违法行为，帮助用户一键向多个监督平台提交投诉。

## 运行环境

- **操作系统**：Windows / macOS / Linux
- **Python 版本**：3.6 及以上
- **依赖**：仅使用 Python 标准库（tkinter），无需安装额外包

## 启动方法

### Windows 用户

双击 `一键投诉.py` 文件，或在命令行中运行：

```
python 一键投诉.py
```

### macOS / Linux 用户

在终端中运行：

```
python3 一键投诉.py
```

### 操作步骤

1. **选择省份**：在顶部下拉框选择您所在的省份，省人大信访和省信访局将自动匹配对应网址
2. **撰写投诉内容**：在左侧文本框中撰写投诉材料（可点击"加载模板"使用预设模板）
3. **选择投诉平台**：在右侧勾选要投诉的平台（默认全选）
4. **一键投诉**：点击底部红色按钮，程序将自动复制内容到剪贴板并打开所有选中的投诉网站
5. **粘贴提交**：在每个投诉页面中按 Ctrl+V 粘贴投诉内容并提交

### 其他功能

- **加载模板**：使用内置的投诉模板，包含法律依据
- **从文件导入**：从 .txt 文件导入投诉内容
- **保存到文件**：将投诉内容保存为 .txt 文件
- **保存/加载草稿**：保存当前编辑状态，下次打开可继续编辑
- **仅复制内容**：只复制投诉内容到剪贴板，不打开网站
- **投诉历史**：自动记录每次投诉操作（保存在用户目录下）

## 法律依据

- 《中华人民共和国民事诉讼法》第一百二十六条
- 《最高人民法院关于人民法院登记立案若干问题的规定》
- 《人民法院工作人员处分条例》

## 注意事项

1. 投诉内容请实事求是，如实反映情况
2. 建议保留好起诉材料的递交凭证（如收据、EMS回执等）
3. 部分投诉平台需要实名注册后才能提交
4. 投诉后请关注各平台的回复和处理进度
5. 本程序仅帮助打开投诉页面并复制内容，实际提交需要用户在各平台手动完成
